About the Project

jQuery-AJAX API Project
Dec 2021 - Current

In this project we were asked to display crypto coins from the coinGecko API. We have a SPA with a route to three different pages. The page displays coin data with an option to get more info on each coin. The client can toggle a list of 5 coins for which he can see graph data on the Live Reports page regarding those 5 coins. The third page is the about me page + the explanation of this project. In this project we used local storage so when the user presses more info he does not have to always make an API call if under 2 minutes.
